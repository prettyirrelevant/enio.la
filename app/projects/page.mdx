# projects

some things i've built.

## [<Neuron height={18} />](https://github.com/prettyirrelevant/final-year-project 'Neuron')

my undergraduate final year project. a student attendance system using bluetooth low energy with an esp32 microcontroller. the system includes a mobile app, an authentication server, and the iot component for handling bluetooth communication.

## [<WrappedNaira height={18} />](https://github.com/prettyirrelevant/wrapped-naira 'Wrapped Naira')

a proof of concept fiat-to-crypto ramp that allows users to convert naira into a stablecoin erc20 token. features a p2p exchange mechanism for seamless on/off-ramping between fiat and crypto.

## [<Gistrunner height={18} />](https://github.com/prettyirrelevant/gistrunner 'Gistrunner')

a sandbox environment for executing code snippets directly from github gists. supports multiple languages with isolated execution environments.

## [<Bridgebloc height={18} />](https://github.com/prettyirrelevant/bridgebloc 'Bridgebloc')

a token bridge for transferring assets across evm compatible chains. built during a hackathon to explore cross-chain interoperability patterns.

## [<Decodify height={18} />](https://github.com/prettyirrelevant/decodify 'Decodify')

a browser extension that decodes etherscan transactions into human-readable format using rotki's transaction decoder. makes it easier to understand what on-chain transactions actually do.

## [<Waakye height={18} />](https://github.com/prettyirrelevant/waakye 'Waakye')

a playlist converter that moves your music between spotify, youtube music, and deezer. handles matching tracks across platforms and preserving playlist metadata.
